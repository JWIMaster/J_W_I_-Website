Swift Toolchain Installation Guide for iOS 6

Install the recommended Xcode version for your macOS and download the corresponding Swift toolchain build file. For Big Sur and Monterey, make sure to download both the primary toolchain and Swift 5.1.5.

macOS Xcode Downloads Notes
Mojave Xcode 11.3.1 Swift 5.1.5 Build Build Script GitHub reference: just download the build file
Big Sur Xcode 13.2.1 Swift 5.6.3 Build Build Script Also download Swift 5.1.5 Build Big Sur requires both 5.6.3 and 5.1.5 for iOS 6 projects.
Monterey Xcode 13.4.1 Swift 5.10.1 Build Build Script Also download Swift 5.1.5 Build Monterey requires both 5.10.1 and 5.1.5 for iOS 6 projects.

Step 2: Install Xcode

Install the Xcode version recommended for your macOS above.

Step 3: Fix Swift Toolchain Paths

sudo install_name_tool -add_rpath /Library/Developer/Toolchains/swift-5.6.3-RELEASE.xctoolchain/usr/lib/swift/macosx /Library/Developer/Toolchains/swift-5.6.3-RELEASE.xctoolchain/usr/bin/swift-driver
sudo install_name_tool -add_rpath /Library/Developer/Toolchains/swift-5.10.1-RELEASE.xctoolchain/usr/lib/swift/macosx /Library/Developer/Toolchains/swift-5.10.1-RELEASE.xctoolchain/usr/bin/swift-driver

Step 4: Install Xcode Templates

mkdir -p ~/Library/Developer/Xcode/Templates/Project\ Templates/Application && git clone https://github.com/JWIMaster/iOS-6-Swift-Xcode-Templates.git /tmp/iOS6SwiftTemplates && cp -R /tmp/iOS6SwiftTemplates/Templates/Project\ Templates/Application/* ~/Library/Developer/Xcode/Templates/Project\ Templates/Application/ && rm -rf /tmp/iOS6SwiftTemplates

Step 5: Select the Legacy Swift Application Template

Open Xcode, create a new project, and choose the “Legacy Swift Application” template. You’re good to go.

How to Build

  1. cd into your project directory
  2. Drag the build file you downloaded onto the terminal
  3. Type your project scheme name after the build file
  4. Specify the build folder (put "build" if unsure)
  5. Enjoy your IPA!
